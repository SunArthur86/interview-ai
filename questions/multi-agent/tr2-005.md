# 在 Multi-Agent 系统中，CrewAI 的“角色扮演”模式与 LangGraph 的“图状态”模式在构建协作逻辑时有什么本质区别？

CrewAI 基于角色扮演的预设逻辑，核心是将 Agent 定义为具有特定角色、目标和背景故事的“员工”，通过顺序或层级的过程来协作，适合结构化明确、分工清晰的业务流（如营销团队模拟）。其逻辑相对固定，依赖预定义的 Task 链。LangGraph 则基于有向循环图（Cyclic Graph）和全局状态机，核心是定义状态如何在节点间流转和转换。它更加灵活，支持复杂的条件跳转、回退和动态分支，适合构建需要根据中间结果动态改变执行路径的应用（如复杂的客服审批流）。简言之，CrewAI 更像排好的剧本，LangGraph 更像一套可编程的逻辑控制器。
