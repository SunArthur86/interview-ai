# Plan-and-Execute（规划与执行）与 ReAct（推理与行动）范式在解决复杂任务时有哪些核心区别？在什么场景下应优先选择 Plan-and-Execute？

ReAct 采用推理与行动的交错迭代，适合探索性任务，但 Token 消耗高且易陷入循环；Plan-and-Execute 采用先规划后执行的分阶段模式，Token 成本低且具备全局视角，适合路径清晰、步骤固定的复杂长链条任务。
