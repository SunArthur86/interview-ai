# Pre-Norm 和 Post-Norm 区别

Pre-Norm (Layer Normalization 在子层之前) 和 Post-Norm (Layer Normalization 在子层之后) 是 Transformer 架构中两种不同的归一化位置策略，它们显著影响了模型的训练稳定性与梯度传播。

**核心区别与原理**：
1. **Post-Norm (原始 Transformer)**：结构为 `Input -> Sub-Layer (Attn/FFN) -> Add & LN -> Output`。残差连接包裹在子层和归一化外侧。
   - **原理**：梯度流向：$\nabla x = \nabla y + \nabla SubLayer(x)$。随着层数加深，$SubLayer$ 的梯度贡献叠加，容易导致梯度爆炸或消失，训练深层网络（如 >12 层）时通常需要 Warmup（学习率预热）来维持稳定。
2. **Pre-Norm (GPT-2/3, LLaMA 等)**：结构为 `Input -> LN -> Sub-Layer -> Add -> Output`。残差连接直接连接输入和输出。
   - **原理**：梯度流向：$\nabla x = \nabla y + \nabla SubLayer(LN(x))$。由于存在直接的“恒等映射”通路（Identity Path），梯度可以无损地流向浅层，有效缓解了深层网络的梯度退化问题。

```text
+-----------------------+ Post-Norm 结构 (原始 Transformer) +-----------------------+
|                                                                             |
|   x_in                                                                        |
|    |                                                                         |
|    +--->[ Sub-Layer (Attn/FFN) ]--->[ Add ]--->[ Layer Norm ]---> x_out       |
|    |                                           |                             |
|    +-------------------------------------------+                             |
|                                                                             |
+-----------------------------------------------------------------------------+

+-----------------------+ Pre-Norm 结构 (GPT/LLaMA 等) +------------------------+
|                                                                             |
|   x_in                                                                        |
|    |                                                                         |
|    +--->[ Layer Norm ]--->[ Sub-Layer (Attn/FFN) ]--->[ Add ]---> x_out      |
|    |                                                   |                     |
|    +---------------------------------------------------+                     |
|                                                                             |
+-----------------------------------------------------------------------------+
```

### 实战深化
- **实战案例**：在尝试复现 GPT-3 或 LLaMA 时，如果不小心沿用原始 BERT 的 Post-Norm 结构并移除 Warmup，深层模型（如 24 层以上）在训练初期 Loss 往往会直接变为 NaN。大多数现代 LLM 切换到 Pre-Norm 后，可以显著缩短 Warmup 步数甚至无需 Warmup 即能稳定收敛。

- **代码示例**：
```python
import torch
import torch.nn as nn

class PreNormBlock(nn.Module):
    def __init__(self, d_model, attn, ff):
        super().__init__()
        self.norm1 = nn.LayerNorm(d_model)
        self.norm2 = nn.LayerNorm(d_model)
        self.attn = attn
        self.ff = ff

    def forward(self, x):
        # Pre-Norm: 先归一化，再进入子层，最后残差相加
        x = x + self.attn(self.norm1(x))
        x = x + self.ff(self.norm2(x))
        return x
```

- **对比表格**：

| 特性 | Pre-Norm | Post-Norm |
| :--- | :--- | :--- |
| **归一化位置** | 子层之前 | 子层之后 |
| **梯度传播** | 极佳（存在恒等映射通路） | 较差（深层易梯度消失）
|n| **训练稳定性** | 高，无需 Warmup 也能收敛 | 低，必须配合 Warmup
| **模型性能** | 收敛快，深层更易训练 | 理论上上限略高（但难训练）
| **代表模型** | GPT-2/3, LLaMA, ViT | BERT, 原始 Transformer |
