# 强化学习中策略函数和值函数

主流的强化学习方法可以划分为免模型学习和有模型学习两类。免模型学习通过直接优化策略或值函数来最大化累积奖励，代表方法有 A3C、PPO、DQN 和其改进算法；有模型学习通过构建环境模型来进行长远规划，代表方法有 MPC 和 Expert Iteration。每种方法在不同的应用场景中有其独特的优势和局限性。

在强化学习中，策略函数和值函数是两个核心的概念，它们分别用于描述智能体如何选择动作以及这些动作的长期回报。

### 策略函数
策略函数 \(\pi(a|s)\) 描述了在给定状态 \(s\) 下，智能体选择某个动作 \(a\) 的概率或确定性规则。
- **随机策略**：输出动作的概率分布，智能体根据概率采样动作。
- **确定性策略**：直接输出确定的动作，\(a = \pi(s)\)。

### 值函数
值函数描述了在某个状态下，按照特定策略行动所能获得的期望累积回报。
- **状态值函数 \(V(s)\)**：从状态 \(s\) 出发遵循策略 \(\pi\) 的期望回报。
- **动作值函数 \(Q(s, a)\)**：在状态 \(s\) 下采取动作 \(a\) 并在此后遵循策略 \(\pi\) 的期望回报。

### 关系
策略函数决定了“怎么做”，值函数评估了“现在好还是未来好”。强化学习的目标通常是优化策略函数以最大化值函数。

### 实战案例
在 OpenAI Five (Dota 2 AI) 中，单独使用基于价值的方法（如 Q-Learning）无法处理巨大的连续动作空间，因此使用了基于策略优化的 PPO；而在 Atari 游戏中，动作空间离散且较小，DQN（基于值函数）效果更好。实际工程中常混用，如 A3C/SAC，利用 Critic 评估方差，降低 Actor 训练难度。

### 关键代码示例
```python
# Actor-Critic 结构简例
class ActorCritic(nn.Module):
    def forward(self, state):
        # 策略函数: 输出动作概率分布
        action_probs = self.actor(state) 
        # 值函数: 评估当前状态价值
        state_value = self.critic(state)
        return action_probs, state_value
```

### 策略函数 vs 值函数对比

| 维度 | 策略函数 | 值函数 |
| :--- | :--- | :--- |
| **核心定义** | 给定状态下采取动作的概率/确定性动作 | 给定状态（或状态-动作对）的期望累积回报 |
| **输出形式** | 离散动作空间为概率分布，连续为具体数值 | 标量值 |
| **主要作用** | 控制 Agent 的行为，决定“怎么做” | 评估当前局势的好坏，指导“往哪走” |
| **典型算法** | REINFORCE, PPO, DPG | DQN, Monte Carlo, TD-Learning |
| **是否需环境模型**| 不需要 | 不需要 |
| **优化目标** | 最大化期望回报 | 最小化 Bellman Error (TD Error) |

## 常见考点
1. **Actor-Critic 架构**：解释策略网络和价值网络如何协同工作，Critic 如何减少 Actor 的方差。
2. **Bellman 方程**：请写出 V(s) 和 Q(s,a) 的 Bellman 期望方程，并解释其递归性质。
3. **探索与利用**：策略函数中 epsilon-greedy 或熵正则化项是如何平衡探索新动作和利用已知高奖励动作的？

## 记忆要点

- 策略函数：决定怎么做，输出状态s下采取动作a的概率分布π(a|s)。
- 值函数：决定往哪走，评估状态或动作的期望累积回报(标量V(s)或Q(s,a))。
- 网络分工：Actor-Critic架构中，Actor输出动作，Critic评估方差降低训练难度。
- 优化目标：策略函数负责控制行为，其优化目标是由值函数提供的长期回报评估。
