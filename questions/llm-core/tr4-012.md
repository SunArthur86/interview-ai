# 大模型知识蒸馏到小模型时，除了 Logits 蒸馏，还有哪些针对特定能力的蒸馏策略？

除了传统的软标签 Logits 蒸馏，针对 LLM 的特点还有以下策略：1. **Chain-of-Thought Distillation**：不仅仅蒸馏最终答案，还强迫小模型学习大模型的推理过程，将思维链作为监督信号。2. **Feature-based Distillation**：提取大模型中间层的隐藏状态，要求小模型的对应层特征与大模型对齐，这有助于迁移模型的结构化知识。3. **Black-box distillation（API 蒸馏）**：在无法获取大模型参数时，利用大模型 API 生成高质量的合成数据来训练小模型。4. **Task-specific Distillation**：针对数学、代码等特定领域，利用大模型生成的专项数据集进行微调。这些策略结合使用，可以在极小参数量下保留大模型的大部分推理能力。
