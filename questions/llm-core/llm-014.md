# GLM的自回归空白填充方法与BERT中的使用遮蔽语言模型(MLM)

### GLM 的自回归空白填充 vs BERT 的 MLM

**核心区别**：
BERT 使用的是**遮蔽语言模型（MLM）**，这是一种自编码方法，通过双向上下文预测被遮蔽的词。而 GLM 采用的是**自回归空白填充**，它在结构上类似于自回归模型（如 GPT），但通过特殊的“填空”任务实现了双向上下文的利用。

**1. BERT (MLM) 机制**
*   **原理**：将输入序列中的部分 Token 替换为 `[MASK]`，模型利用**双向注意力**机制同时看到被遮蔽词的左右两侧上下文来预测原词。
*   **特点**：由于使用了 Mask 和双向注意力，BERT 在自然语言理解（NLU）任务上表现优异。
*   **局限**：
    *   **预训练与微调差异**：预训练时的 `[MASK]` 标记在微调时不存在，造成数据分布差异。
    *   **生成能力弱**：假设被遮蔽词之间相互独立（实际上是联合预测），难以处理生成任务。

**2. GLM (自回归空白填充) 机制**
*   **原理**：
    1.  **填空任务**：在文本中随机遮蔽一段连续的 Token（跨度 Masking），形成空白。
    2.  **自回归生成**：模型以自回归（从左到右）的方式生成这段空白的内容。模型在预测空白处的下一个词时，可以看到左边（上下文）和右边（未遮蔽的后续文本）的信息。
    3.  **2D 位置编码**：使用二维位置编码来区分原文位置和生成位置，保证模型知道自己在填哪个空。
*   **特点**：虽然是自回归生成，但因为在填空时能利用右侧的上下文信息，它实际上也具备了双向理解能力。

**架构对比示意（ASCII）：**

```text
BERT (MLM) 结构：
Input:  [CLS] The cat sat on the [MASK] [SEP]
Attention: <---双向注意力--->
Output: Predict "mat" (独立预测，无顺序依赖)

GLM (Autoregressive Blank Infilling) 结构：
Input (Corrupted):  The cat [MASK] [MASK] the mat
Input (Reordered):  [gMASK] The cat sat on the mat  (填空)
Attention Flow:    [Prompt] <---(看右边)---> [Blank_1] [Blank_2] (自回归生成)
Output:            "sat"      "on"
```

**GLM 在 NLU 任务中的优势**：
1.  **统一理解与生成**：GLM 将 NLU 任务转化为生成任务（例如分类任务转化为“生成类别标签”），使得同一个模型可以同时处理理解和生成任务。
2.  **更好的上下文建模**：相比于 BERT 独立预测每个 Mask 词，GLM 对空白处的词进行连续的序列生成，能更好地建模词与词之间的依赖关系（如长距离依赖）。
3.  **无预训练微调差异**：在微调阶段，GLM 只需要调整 Prompt 或输入格式，不需要引入特殊的 `[MASK]` 标记，更符合自然语言生成范式。

## 常见考点
1. **GLM 如何解决自回归模型无法看到未来信息的问题？**
   - GLM 通过重排输入序列，将待填充的空白部分移到序列末尾（或通过特殊的 Attention Mask 机制），使得模型在生成空白处的词时，能够 Attention 到原始输入中被遮蔽部分两侧的上下文（即未遮蔽的后文也是可见的），实现了单向生成过程中的双向上下文利用。
2. **GLM 与 BERT 在 Mask 方式上有什么不同？**
   - BERT 通常是随机 Mask 单个 Token；GLM 采用 Span Masking，随机 Mask 连续的一段文本片段，并使用自回归的方式恢复这段片段。
3. **什么是 2D 位置编码（2D Positional Encoding）？**
   - 为了让模型区分“原始输入的位置”和“生成填空的位置”，GLM 使用了二维位置编码。通常包含两个向量：一个表示该词在原始文本中的位置，另一个表示该词在生成序列中的位置（如果是被 Mask 的部分）。

## 记忆要点

- BERT（MLM）：双向注意力独立预测Mask词，擅长理解。
- GLM：自回归填空，Span Masking，按顺序生成空白内容。
- GLM通过2D位置编码和Attention Mask实现生成时看右侧上下文。
- GLM统一了理解和生成任务，无预训练微调差异。
