# DeepSpeed ZeRO系列原理

DeepSpeed ZeRO (Zero Redundancy Optimizer) 是一种针对大模型训练的显存优化技术，旨在消除数据并行中的冗余存储。

### 核心思想
在标准数据并行（DDP）中，每个 GPU 都保存完整的模型参数、梯度和优化器状态，这造成了极大的冗余。ZeRO 通过切分这些状态，在保持计算逻辑不变的前提下，大幅降低单卡显存占用。

### 三个阶段

#### 1. ZeRO-1 (Optimizer State Partitioning)
- **切分对象**：优化器状态（如 Adam 的一阶矩和二阶矩）。
- **做法**：将优化器状态按维度平均切分到不同 GPU 上。
- **效果**：显存占用减少约 4 倍（假设使用 Adam）。

#### 2. ZeRO-2 (Gradient Partitioning)
- **切分对象**：优化器状态 + 梯度。
- **做法**：在反向传播计算梯度后，不立即同步更新，而是将梯度切分存储并更新对应部分的参数。通过 All-Gather 获取完整参数进行前向计算，但不需要在每卡上都存全量梯度。
- **效果**：显存占用进一步减少，约为 DDP 的 1/8。

#### 3. ZeRO-3 (Parameter Partitioning)
- **切分对象**：优化器状态 + 梯度 + 模型参数。
- **做法**：模型参数也被切分到不同 GPU 上。当某一层需要计算时，动态地从其他 GPU 收集（All-Gather）该层的参数，计算完成后立即释放。
- **效果**：显存占用极低，线性扩展能力极强，支持训练万亿参数模型。

### 实战案例
在训练 175B 模型时，如果不开启 ZeRO-3，单卡 A100（80GB）连加载模型都做不到；开启 ZeRO-3 Offload 后，甚至可以在单张卡上利用 CPU 内存和 NVMe 完成训练（尽管速度较慢）。常见坑点是设置 `offload_optimizer` 后，训练吞吐量可能下降 30%-50% 需权衡。

### ZeRO vs DDP 显存占用对比

```text
DDP (Data Parallel):
GPU 0: [Params][Grads][Optimizer States] (Full Copy)
GPU 1: [Params][Grads][Optimizer States] (Full Copy)
GPU 2: [Params][Grads][Optimizer States] (Full Copy)

ZeRO-3:
GPU 0: [Params_1/3][Grads_1/3][Opt_1/3] 
GPU 1: [Params_2/3][Grads_2/3][Opt_2/3]
GPU 2: [Params_3/3][Grads_3/3][Opt_3/3]
(Forward/Backward: All-Gather params temporarily)
```

### DDP 与 ZeRO 各阶段对比

| 特性 | DDP | ZeRO-1 | ZeRO-2 | ZeRO-3 |
| :--- | :--- | :--- | :--- | :--- |
| **参数存储** | 每卡全量 | 每卡全量 | 每卡全量 | **分片存储** |
| **梯度存储** | 每卡全量 | 每卡全量 | **分片存储** | 分片存储 |
| **优化器状态** | 每卡全量 | **分片存储** | 分片存储 | 分片存储 |
| **通信量** | 低 (AllReduce) | 低 | 中 (Reduce-Scatter) | 高 (All-Gather + Reduce-Scatter) |
| **计算逻辑** | 未改变 | 未改变 | 未改变 | 未改变 (动态 Gather) |
| **适用场景** | 模型能装下单卡 | 显存吃紧，瓶颈在 Opt | 显存更吃紧 | **超大规模模型** |

### 优势
- **显存节省**：支持在有限显存上训练超大模型。
- **通信优化**：虽然增加了通信量，但通过计算与通信重叠，维持了较高的计算效率。

## 常见考点
1. **通信开销分析**：ZeRO-3 引入了大量的 All-Gather 和 Reduce-Scatter 通信，在什么情况下（如模型极大、显卡带宽受限）会比模型并行（TP）更优？
2. **Offload 机制**：DeepSpeed ZeRO-Offload 如何将优化器状态卸载到 CPU 内存，以及这会对训练速度产生多大影响？
3. **ZeRO-Infinity**：这是针对 ZeRO-Offload 的进一步优化，利用 NVMe 进行更极致的显存卸载，其核心原理是什么？
