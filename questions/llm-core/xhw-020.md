# WordPiece如何考虑语言模型概率

WordPiece 在合并子词时，不仅考虑频率，还引入了**语言模型概率**来优化选择。

**1. 核心机制**
与 BPE 直接选择频率最高的符号对不同，WordPiece 的目标是最大化训练语料的**似然概率**。即在每一步合并中，选择能够使训练数据整体概率增加最多的那个子词对。

**2. 评分准则**
假设语料库由一系列子词序列组成，WordPiece 试图最大化如下对数似然：
$$ \log P(w) = \sum_{i} \log P(subword_i) $$
在合并候选 $(x, y)$ 为新词 $z$ 时，计算合并前后的概率变化（通常简化为新词 $z$ 的频率与其对数概率的乘积，即 score）。公式类似于：
$$ score = \frac{freq(z)}{freq(x) \cdot freq(y)} $$
（注：实际实现中通常取 Log 形式，选择使 $\sum \log P(w)$ 最大的合并）。
*   **细节补充**：这里的 $P(subword)$ 实际上是基于当前词表训练的一元语言模型概率。如果 $x$ 和 $y$ 经常一起出现，合并后的 $freq(z)$ 较高，且 $freq(x) \cdot freq(y)$ 相对较低（独立性弱），则 Score 最高。

**3. 直观理解**
它倾向于合并那些**结合后产生语义**且**降低整体语言模型困惑度**的词对。相比于单纯贪心的频率统计，这种方法能产生更符合语言构词法的词汇表（例如倾向于合成有意义的词根词缀）。

**4. 实战补充**
*   **实战案例**：在使用 BERT Base 模型进行金融领域文本微调时，发现大量专业术语（如“DeFi”）被切分得过碎（如“De”, “##Fi”），导致语义理解偏差。通过在领域语料上重新运行 WordPiece 算法，利用其语言模型特性合并这些高频且语义明确的词对，显著提升了模型对该领域术语的表征能力。
*   **代码示例**：使用 `tokenizers` 库训练 WordPiece 模型的核心逻辑：
```python
from tokenizers import trainers, Tokenizer, models

# 初始化 WordPiece 模型
tokenizer = Tokenizer(models.WordPiece(unk_token="[UNK]"))

# 定义训练器（相比 BPE，WordPiece 侧重特殊的评分）
trainer = trainers.WordPieceTrainer(
    vocab_size=30000,
    special_tokens=["[UNK]", "[CLS]", "[SEP]", "[PAD]", "[MASK]"],
    # WordPiece 内部会基于似然概率而非单纯频率来合并
)
tokenizer.train(["path/to/corpus.txt"], trainer=trainer)
```

**5. BPE vs WordPiece 对比**

| 特性 | BPE (Byte Pair Encoding) | WordPiece |
| :--- | :--- | :--- |
| **合并依据** | 基于统计共现频率 | 基于语言模型似然概率 (LM Probability) |
| **评分公式** | $Score = Freq(x, y)$ | $Score = \frac{Freq(z)}{Freq(x)Freq(y)}$ (最大化似然) |
| **优化目标** | 贪心减少词表长度/步数 | 最大化训练数据的联合概率 |
| **生成词表** | 倾向于合并高频词，可能包含无意义组合 | 倾向于保留语义完整性，避免拆分有意义的词 |
| **典型应用** | GPT-2, GPT-3, RoBERTa | BERT, DistilBERT |

## 常见考点
1.  **与 BPE 的区别**：BPE 依据共现频率，WordPiece 依据似然概率。BPE 是贪心合并频率最高的对，WordPiece 是合并最能提升语言模型概率的对。通常效果相近，但 WordPiece 略偏向于有语义的词。
2.  **Unigram LM**：WordPiece 和 Unigram 语言模型分词法的区别？（Unigram 初始有一个大词表，通过不断删除似然增益最小的词来精简词表；WordPiece 是从字符开始不断合并）。
3.  **应用场景**：WordPiece 最著名的应用模型是？（BERT 系列模型主要使用 WordPiece）。

## 记忆要点

- 核心准则：BPE贪心选共现频率最高的，WordPiece选似然概率增益最大的。
- 合并公式：WordPiece评分看 Score=freq(xy)/(freq(x)*freq(y))，目标是最大化语料的联合概率。
- 因为基于概率模型，所以WordPiece比BPE更能保留词根词缀的语义完整性(如BERT)。
