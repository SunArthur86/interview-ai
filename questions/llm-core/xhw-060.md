# Group/Instance Normalization

### Group Normalization (GN) vs Instance Normalization (IN)

#### 1. Group Normalization (GN)
- **定义**：将通道分成若干组，在每个组内（包含空间维度 H, W）计算均值和方差进行归一化。
- **核心特点**：
  - **不依赖 Batch Size**：这是相对于 BN 的最大优势，适合小 Batch 训练（如目标检测、分割）。
  - **分组数 G 可调**：当 $G=1$ 时类似 Layer Norm（跨通道统计）；当 $G=C$ 时即为 Instance Norm。
- **计算公式**：
  设输入为 $(N, C, H, W)$， reshape 为 $(N, G, C/G, H, W)$。
  计算均值和方差在 $(C/G, H, W)$ 维度上进行。

#### 2. Instance Normalization (IN)
- **定义**：对**每个样本的每个通道**单独计算均值和方差进行归一化。即 $G=C$ 的特殊情况。
- **核心特点**：
  - **风格迁移首选**：它去除了样本间的统计差异（如对比度和亮度信息），保留了图像的风格特征，使得生成的图像风格一致性更好。
  - **忽略全局统计**：对于分类任务可能不利，因为它剥离了用于判别的全局统计信息。
- **细节**：对于每个 Channel $c$ 和样本 $n$，只在 $H \times W$ 空间维度计算均值方差。

**计算维度可视化**：
```text
输入 Tensor Shape: (N, C, H, W)

Batch Norm (BN):
统计量聚合维度: [N, H, W]
结果 Shape: (N, C, 1, 1)  <- 每个Channel有一个统计量

Layer Norm (LN):
统计量聚合维度: [C, H, W]
结果 Shape: (N, 1, 1, 1)  <- 每个Sample有一个统计量

Instance Norm (IN):
统计量聚合维度: [H, W]
结果 Shape: (N, C, 1, 1)  <- 每个Sample的每个Channel一个统计量

Group Norm (GN) (Groups=G):
统计量聚合维度: [C/G, H, W]
结果 Shape: (N, G, 1, 1)  <- 每个Sample的每组Channel一个统计量
```

#### 3. 区别总结
| 特性 | Group Norm (GN) | Instance Normalization (IN) |
| :--- | :--- | :--- |
| 计算维度 | (N, Group, H, W) | (N, Channel, H, W) 即每个通道独立 |
| 主要用途 | 通用 CV 任务，特别是小 Batch 场景 | 风格迁移，生成式任务 |
| 对 Batch 依赖 | 无 | 无 |
| 共享信息 | 组内通道共享统计信息 | 通道间完全独立 |

#### 4. 实战深化

**实战案例**：
在目标检测模型（如 Mask R-CNN）训练中，当 GPU 显存受限导致 Batch Size 只能设为 2 时，使用 BN 会导致统计量估计严重失真，模型 Loss 震荡不收敛；切换至 Group Norm（设置 G=32）后，训练曲线立即恢复平滑，且无需同步 Batch Size。

**代码示例 (PyTorch)**：
```python
import torch
import torch.nn as nn

# 输入: (Batch=2, Channel=64, H=32, W=32)
x = torch.randn(2, 64, 32, 32)

# Group Norm: 将64个通道分为32组，每组2个通道计算统计量
gn = nn.GroupNorm(num_groups=32, num_channels=64)
out_gn = gn(x)
print(f"GN Output Shape: {out_gn.shape}") # 保持 [2, 64, 32, 32]

# Instance Norm: 相当于 Group Norm where num_groups=num_channels
in_layer = nn.InstanceNorm2d(64)
out_in = in_layer(x)
print(f"IN Output Shape: {out_in.shape}")
```

## 记忆要点

- GN将通道分组计算统计量，不依赖Batch Size，适合小Batch训练如目标检测。
- IN是GN的特例(G=C)，对每个样本的每个通道独立归一化，常用于风格迁移。
- GN分组数G可调：G=1类似LayerNorm，G=C即为InstanceNorm。
