# 解释一下P-tuning

### P-tuning (含 P-tuning v2)

**概念**：
P-tuning 是一种专注于优化连续提示的参数高效微调方法。它通过在输入文本前插入一段可训练的连续向量（即“软提示”），代替手工设计的离散文本提示，来提升模型在下游任务中的性能。

**来源**：
*   P-tuning: Liu et al. (2021), 《GPT Understands, Too》
*   P-tuning v2: Liu et al. (2021), 《P-tuning v2: Prompt tuning can be comparable to fine-tuning universally across scales and tasks》

**核心原理**：
1.  **连续提示**：将原本离散的自然语言提示词转换为连续的嵌入向量空间中的参数。这些向量在训练过程中通过反向传播自动优化，寻找最能激活模型能力的“提示”。
2.  **P-tuning v2 的改进**：原始 P-tuning 仅在输入层添加提示，在参数量较小或任务较难时表现不如全量微调。P-tuning v2 借鉴了 Prefix-tuning 的思想，在 Transformer 的**每一层**都添加了可训练的连续提示，并将其应用于重采样机制，使其在各类规模模型和任务（包括 NLU 和 NLG）上都能媲美全量微调。

**技术细节**：
*   **提示嵌入**：在输入序列 $X$ 前拼接可训练向量 $P$，得到 $[P; X]$。这些向量不需要对应真实的单词。
*   **MLP 提示编码器 (v1)**：原始 P-tuning 使用一个 LSTM 或 MLP 对提示向量进行编码，以增加提示的复杂度和表达能力，解决深层模型中连续提示训练不稳定的问题。
*   **多层微调 (v2)**：在每一层的 Key、Value 矩阵计算前都加入可训练的 Prompt 向量（类似 Prefix-tuning），但在结构上做了简化，移除了重参数化中的 MLP（在 v2 中），直接优化 Prompt Embedding，使其在不同层间共享或独立。

**架构对比图：**
```text
P-tuning v1:
Input: [Prompt (MLP Encoded)] [Text Tokens]
  |
  +--> Layer 1 --> Layer 2 --> ... --> Output

P-tuning v2:
Input: [Text Tokens]
  |
  v
Layer 1: [Prompt] + Text ---> FeedForward
  |
  v
Layer 2: [Prompt] + Layer1_Out ---> FeedForward
  |
  ...
```

**优势**：
*   **灵活性**：连续向量可以表示离散语言难以描述的复杂特征，打破了人工设计的词汇限制。
*   **通用性 (v2)**：解决了 Prompt Tuning 难以在小模型（如 BERT-base）上表现良好的问题，适应各种规模模型和任务类型（特别是分类等 NLU 任务）。

**局限**：
*   **占用上下文长度**：无论是 v1 还是 v2，如果依赖于输入层的拼接，都会占用额外的输入序列长度空间，减少了有效输入长度。
*   **超参数敏感**：提示长度、每一层是否添加 Prompt 等超参数对性能有一定影响，需要调优。

## 常见考点
1. **P-tuning v2 vs Prefix-tuning**：两者非常相似，都在每一层加 Prompt。主要区别在于 P-tuning v2 去掉了 Prefix-tuning 中的重参数化（MLP），直接对 Prompt Embedding 进行微调，并且在某些层（如 Embedding 层）的处理上略有不同，旨在适配更广泛的任务（尤其是 NLU）。
2. **Hard Prompt vs Soft Prompt**：P-tuning 属于 Soft Prompt（连续向量），传统离散文本属于 Hard Prompt。Soft Prompt 难以人工解释，但在性能上限上通常更高。
3. **为什么 v1 在小模型上表现差**：因为小模型对上下文的学习能力较弱，仅靠输入层的连续提示难以引导深层特征的变化。v2 通过在每一层都加 Prompt，解决了这个问题。

## 记忆要点

- P-tuning v1：仅在输入层加连续Prompt，用MLP编码。
- P-tuning v2：在每一层都加Prompt，类似Prefix Tuning但去掉了MLP。
- 核心是将离散文本提示转为可优化的连续向量（软提示）。
- v2解决了小模型和NLU任务表现差的问题，通用性更强。
