# 什么是Medusa解码？它与传统的投机解码相比有什么独特优势？

Medusa解码是一种非自回归并行解码技术，通过在主模型上增加多个预测未来的“输出头”来并行生成候选token，并共享底层Transformer参数。与依赖小模型草拟的传统投机解码相比，Medusa消除了对独立Draft Model的依赖，避免了因小模型能力弱导致的接受率低问题，且单模型部署更简单、显存开销更低，通常能实现更高的加速比。
