# 手撕Normalization

### LayerNorm 实现

LayerNorm 对单个样本的所有特征进行归一化，常用于 NLP 领域（如 Transformer）。它不依赖 Batch 大小。

```python
import torch
from torch import nn

class LN(nn.Module):
    def __init__(self, normalized_shape, eps=1e-5, elementwise_affine=True):
        super().__init__()
        self.normalized_shape = normalized_shape
        self.eps = eps
        self.elementwise_affine = elementwise_affine
        if self.elementwise_affine:
            self.gain = nn.Parameter(torch.ones(normalized_shape)) # gamma (scale)
            self.bias = nn.Parameter(torch.zeros(normalized_shape)) # beta (shift)

    def forward(self, x):
        # 计算归一化维度，通常对最后一个或几个维度求均值
        dims = [-(i + 1) for i in range(len(self.normalized_shape))]
        mean = x.mean(dim=dims, keepdim=True)
        # unbiased=False 表示使用有偏估计（除以N），PyTorch LayerNorm 默认行为
        var = x.var(dim=dims, keepdim=True, unbiased=False) 
        x_norm = (x - mean) / torch.sqrt(var + self.eps)
        if self.elementwise_affine:
            x_norm = self.gain * x_norm + self.bias
        return x_norm
```

### RMSNorm 实现

RMSNorm (Root Mean Square Layer Normalization) 是 LayerNorm 的一种变体，去掉了均值中心化的重校准步骤。它在 LLaMA 等大模型中被广泛使用，因为其在保持性能的同时减少了计算量。

核心公式：$\bar{x}_i = \frac{x_i}{\sqrt{\frac{1}{n} \sum_{j=1}^{n} x_j^2} + \epsilon} \cdot \gamma_i$

```python
class RMSNorm(nn.Module):
    def __init__(self, normalized_shape, eps=1e-5, elementwise_affine=True):
        super().__init__()
        self.normalized_shape = normalized_shape
        self.eps = eps
        self.elementwise_affine = elementwise_affine
        if self.elementwise_affine:
            self.gain = nn.Parameter(torch.ones(normalized_shape)) # gamma
            # RMSNorm 通常没有 beta (bias)，因为它去掉了均值项

    def forward(self, x):
        dims = [-(i + 1) for i in range(len(self.normalized_shape))]
        # 计算均方根
        ms = (x ** 2).mean(dim=dims, keepdim=True)
        x_norm = x * torch.rsqrt(ms + self.eps)
        if self.elementwise_affine:
            x_norm = self.gain * x_norm
        return x_norm
```

### 实战案例
*   **LayerNorm**：在早期的BERT微调任务中，如果将LayerNorm的位置放在残差连接之后（Post-LN），训练深网络时容易出现梯度不稳定，通常需要Warmup。
*   **RMSNorm**：在LLaMA/Mistral等现代大模型中，普遍采用RMSNorm且通常置于残差连接之前，这种去中心化的设计不仅推理速度更快（减少了减法操作），在大规模预训练中收敛也更稳定。

### 对比表格

| 维度 | LayerNorm | RMSNorm |
| :--- | :--- | :--- |
| **计算公式** | $(x - \mu) / \sqrt{\sigma^2 + \epsilon} \cdot \gamma + \beta$ | $x / \sqrt{mean(x^2) + \epsilon} \cdot \gamma$ |
| **是否去均值** | 是 (Centering) | 否 |
| **可学习参数** | $\gamma, \beta$ | $\gamma$ (通常无 $\beta$) |
| **计算量** | 较高 (需计算均值和方差) | 较低 (仅需计算均方根) |
| **适用场景** | 传统 Transformer (BERT, GPT-2/3) | 现代高效大模型 (LLaMA, PaLM) |
