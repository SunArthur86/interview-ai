# 手撕MHA+KV Cache

- **手撕 MHA + KV Cache**:
  - **核心逻辑**:
    1. 初始化全零 `cache_k`, `cache_v` (最大长度)。
    2. Forward中计算当前步的 `xk`, `xv`。
    3. 更新缓存: `cache_k[:, start_pos:...] = xk`。
    4. 拼接历史Key/Value进行Attention计算。
  - *数据流图*:
    ```text
    输入 x_t (当前 token)
       │
       ├──────────────────┐
       ▼                  ▼
    [Wq, Wk, Wv] 投影   RoPE 位置编码
       │                  │
       ▼                  ▼
    q_t (当前)          k_t, v_t (当前)
       │                  │
       │                  ▼
       │           ┌───────────────┐
       │           │ KV Cache 更新 │
       │           │ [k_hist | k_t] │
       │           │ [v_hist | v_t] │
       │           └───────┬───────┘
       │                   │
       └───────┬───────────┘
               ▼
         Attention(Q, K_cache, V_cache)
               │
               ▼
           Output y_t
    ```
  - **实战案例**: 在PagedAttention实现中，KV Cache往往不连续，需通过`block_tables`索引物理显存块，否则无法处理变长序列导致显存浪费。
  - **代码**:
```python
class Attention(nn.Module):
    def __init__(self, args):
        super().__init__()
        # 初始化KV Cache
        self.cache_k = torch.zeros((args.max_batch_size, args.max_seq_len, self.n_heads, self.head_dim)).cuda()
        self.cache_v = torch.zeros((args.max_batch_size, args.max_seq_len, self.n_heads, self.head_dim)).cuda()
        # ... 投影层 wq, wk, wv, wo 初始化 ...

    def forward(self, x, start_pos, freqs_cis, mask):
        bsz, seqlen, _ = x.shape
        # 1. 投影并重塑
        xq, xk, xv = self.wq(x), self.wk(x), self.wv(x)
        xq = xq.view(bsz, seqlen, self.n_heads, self.head_dim)
        xk = xk.view(bsz, seqlen, self.n_heads, self.head_dim)
        xv = xv.view(bsz, seqlen, self.n_heads, self.head_dim)

        # 2. 旋转位置编码
        xq, xk = apply_rotary_emb(xq, xk, freqs_cis)

        # 3. 更新 KV Cache
        self.cache_k[:bsz, start_pos : start_pos + seqlen] = xk
        self.cache_v[:bsz, start_pos : start_pos + seqlen] = xv

        # 4. 拼接历史 (Keys, Values)
        keys = self.cache_k[:bsz, :start_pos + seqlen]
        values = self.cache_v[:bsz, :start_pos + seqlen]

        # 5. Attention计算 (FlashAttention 优化点)
        scores = (torch.matmul(xq, keys.transpose(2, 3)) * self.scale).float()
        if mask is not None:
            scores = scores + mask  # 广播加掩码
        scores = F.softmax(scores, dim=-1).type_as(xq)
        output = torch.matmul(scores, values)  # (bsz, seqlen, n_heads, head_dim)
        return self.wo(output.flatten(-2)) # 输出投影
