# Transformer基本结构和原理

Transformer 是一种完全基于注意力机制的神经网络，主要分为 Encoder 和 Decoder 两部分。

**1. 核心组件**
- **自注意力机制**：通过 Query、Key、Value 计算序列中词与词的相关性，捕捉长距离依赖。
- **前馈神经网络 (FFN)**：对每个位置的向量进行非线性变换。
- **位置编码**：由于没有循环结构，注入位置信息以保留序列顺序。
- **残差连接与层归一化**：稳定深层网络的训练。

**2. 结构分工**
- **Encoder**：由多层自注意力+FFN 组成，输入序列并行处理，提取深层语义特征。
- **Decoder**：包含掩码自注意力（防止看到未来词）、Encoder-Decoder Attention（关注输入）、FFN。用于自回归生成。

**3. 优势**
- 并行计算能力强，训练效率高。
- 能够有效捕捉长距离依赖。

**4. 关键细节与原理补充**
- **多头注意力**：
    - 将 $d_{model}$ 维度的向量切分为 $h$ 个头，每个头在不同子空间计算注意力。例如 $d_{model}=512, h=8$，每个头维度为 64。
    - 公式：$MultiHead(Q,K,V) = Concat(head_1, ..., head_h)W^O$，其中 $head_i = Attention(QW_i^Q, KW_i^K, VW_i^V)$。
    - 作用：让模型能够同时关注不同位置的不同子空间信息（如一个头关注语法结构，另一个头关注语义关联）。
- **Layer Norm（层归一化）位置**：
    - **Pre-Norm**（如 GPT-2, LLaMA）：Norm 在子层输入之前。训练更稳定，适合深层网络，利于梯度回传。
    - **Post-Norm**（原始 Transformer）：Norm 在子层输出+残差之后。需要 Warmup。
- **Positional Encoding（位置编码）**：
    - **Sinusoidal（正弦/余弦）**：原始论文使用，利用三角函数的相对位置性质，具有外推性（可处理比训练时更长的序列）。
    - **Learned（可学习）**：如 BERT，直接学习位置向量，但外推性差。
    - **RoPE（旋转位置编码）**：目前主流（LLaMA），通过旋转矩阵将位置信息注入 Query 和 Key，具有更好的相对位置感知和外推性。

**5. 实战案例与代码**
- **实战案例**：在微调 LLaMA 时，如果不使用 Pre-Norm 结构，梯度很容易在深层网络中消失；同时，若直接截断超过训练长度的序列，模型性能会断崖式下跌，引入 ALiBi 或 RoPE 的插值技术可缓解此问题。
- **代码示例 (Scaled Dot-Product)**：
```python
import torch
import torch.nn.functional as F
# 简化的自注意力核心计算
def scaled_dot_product(q, k, v, mask=None):
    # q, k, v shape: [batch_size, heads, seq_len, d_k]
    attn_scores = torch.matmul(q, k.transpose(-2, -1)) / (k.size(-1) ** 0.5)
    if mask is not None:
        attn_scores = attn_scores.masked_fill(mask == 0, -1e9) # 填充极小值
    return torch.matmul(F.softmax(attn_scores, dim=-1), v)
```

**6. 组件对比表**

| 组件 | Encoder (BERT风格) | Decoder (GPT风格) | 关键作用 |
| :--- | :--- | :--- | :--- |
| **Self-Attention** | 全局可见 | 带因果掩码 | 捕捉上下文关联 |
| **Cross-Attention** | 无 | 有 (仅Enc-Dec架构) | 融合源输入信息 |
| **FFN** | 2层线性+GELU | 同左 | 特征变换与非线性 |
| **Norm位置** | 通常Post-Norm | 通常Pre-Norm | 稳定深层梯度 |

## 记忆要点

- 核心三件套：自注意力(QKV计算关联)、FFN(非线性特征变换)、位置编码(注入顺序信息)。
- 多头机制：因为切分多子空间，所以能同时捕捉不同维度的语法和语义特征。
- 结构对比：Pre-Norm(现主流LLM)梯度稳定易训练深层，Post-Norm(原版)需Warmup防爆炸。
