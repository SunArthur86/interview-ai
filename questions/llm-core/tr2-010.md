# 什么是 LongLoRA？它在微调长文本大模型时相比标准 LoRA 做了哪些关键改进以降低显存开销？

LongLoRA 是一种旨在高效微调长上下文大模型的技术。标准 LoRA 虽然减少了可训练参数，但长上下文带来的注意力机制计算量（显存占用）依然巨大。LongLoRA 提出了两个关键改进：1) **短上下文注意力（Short-Context Attention）**：在训练时将长上下文分成若干组，组内使用稀疏注意力或仅关注局部上下文，从而大幅降低显存峰值；2) **移除组查询注意力（GQA）的限制**：通过改进训练策略，使得原本被冻结的 GQA 参数也可以被微调，从而在保持推理速度的同时提升长文效果。推理时则切换回标准的全注意力机制，无需修改模型结构，从而实现了低成本的长文本微调。
