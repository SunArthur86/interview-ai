# 大模型的训练过程分为哪几个阶段?每个阶段的目标是什么

- **大模型训练三阶段**

1. **预训练**
  - 目标:从海量文本中学习语言规律和世界知识
  - 数据:万亿级 token(网页、书籍、代码)
  - 方法:预测下一个 token
  - 产出:基座模型

2. **指令微调 (SFT)**
  - 目标:让模型学会遵循人类指令
  - 数据:高质量指令-回答对(几万到百万条)
  - 产出:对话模型

3. **对齐**
  - 目标:让回答更安全、有用、符合人类偏好
  - 数据:偏好排序数据(好回答 vs 差回答)
  - 产出:对齐模型

- **一句话理解**:预训练→学知识,SFT→学指令,RLHF→学品味.

- **实战案例**：在一个金融垂类模型项目中，我们发现直接跳过 SFT 进行 RLHF 会导致模型虽然“安全”但“听不懂话”，无法正确输出 JSON 格式。后续先投入 5 万条高质量的金融指令数据进行 SFT，再进行对齐，最终模型在任务完成度上提升了 40%。

- **训练阶段对比表**：
| 特性 | 预训练 (PT) | 指令微调 (SFT) | 对齐 (RLHF/DPO) |
| :--- | :--- | :--- | :--- |
| **核心目标** | 学习通用知识和语言能力 | 学会听指令、完成特定任务 | 对齐人类偏好，确保安全有用 |
| **数据规模** | 万亿级 Token | 几万至百万条样本 | 几万至十万条偏好对 |
| **数据质量** | 侧重多样性和广度 | 侧重指令清晰性和准确性 | 侧重人类反馈的真实性 |
| **训练成本** | 极高 (数千 GPU 卡) | 中等 (几十张卡) | 较高 (需训练 RM 或多轮 PPO) |
| **产出模型** | Base Model (续写能力) | Chat Model (对话能力) | Aligned Model (最终产品) |
